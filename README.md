# 测试项目

## 龙神录
  用C++，SDL加lua来重写龙神录弹幕游戏
### 环境变量
SDL的环境变量设置
SDL_IncludePath:  
例子: "**D:\_SDL2-2.0.10\include**"  
SDL_LibraryPath:  
例子: "**D:\_SDL2-2.0.10\lib**"  
SDL_Image_IncludePath:  
例子: "**D:\_SDL2_image-2.0.5\include**"  
SDL_Image_LibraryPath:  
例子: "**D:\_SDL2_image-2.0.5\lib**"
